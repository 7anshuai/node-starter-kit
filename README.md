# node-starter-kit

Node.js api starter kit (Node.js, Express, GraphQL, Sequelize, Babel, Eslint, Flow, Jest)

## Getting Started

Follow the [getting started guide](./docs/getting-started.md) to download and run the project
([Node.js](https://nodejs.org/) >= 8.16.2)
